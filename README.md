# Image Analysis and Pattern Recognition : project

In order to run the code for project we first need to add the dataset:
- Import the dataset from [Kaggle][https://www.kaggle.com/competitions/iapr-26-uno-vision-challenge/data]
- Unzip the folder "iapr-26-uno-vision-challenge" and add it in this folder (resulting in IAPR26/iapr-26-uno-vision-challenge)
